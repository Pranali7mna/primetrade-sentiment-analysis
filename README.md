Bitcoin Market Sentiment and Trader Performance Analysis

Overview

This project analyzes the relationship between Bitcoin market sentiment and trader performance using the Bitcoin Fear & Greed Index and Hyperliquid historical trading data.

The goal is to identify how market sentiment impacts trader profitability, win rates, trading activity, and overall trading behavior.

Datasets

Bitcoin Fear & Greed Index
- Date
- Classification (Extreme Fear, Fear, Neutral, Greed, Extreme Greed)
- Sentiment Score

Hyperliquid Historical Trading Data
- Account
- Coin
- Execution Price
- Size USD
- Closed PnL
- Fee
- Timestamp
- Trade Direction

Key Insights

- Extreme Greed showed the highest average profitability.
- Extreme Greed achieved the highest win rate among all sentiment categories.
- Fear periods recorded the highest trading activity.
- HYPE was the most traded asset.
- @107 generated the highest cumulative profit.
- Trading fees were highest during Fear periods.
